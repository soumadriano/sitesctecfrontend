## 🧠 O que estou (re)aprendendo

### Front-End
- Estrutura semântica com HTML5
- Estilização e responsividade com CSS3 (Flexbox, Grid)
- Interatividade com JavaScript (DOM, eventos, fetch)

### Back-End
- Conceitos de cliente-servidor
- Introdução a Node.js (ou outra linguagem adotada no curso)
- Rotas, requisições HTTP e primeiras APIs
- Persistência de dados (SQL ou MongoDB)
